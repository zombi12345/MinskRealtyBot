# Minsk Realty Bot 🤖

Telegram-бот для поиска квартир в Минске с анализом инфраструктуры.

## Команды
- `/start` - начать работу
- `/all` - показать все квартиры

## Примеры запросов
- `1 комнату до 70000`
- `2 комнаты до 90000`
- `все квартиры`

## Деплой на Render
1. Форкните репозиторий
2. Создайте Web Service на Render
3. Подключите GitHub
4. Нажмите Deploy
